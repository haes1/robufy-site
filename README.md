# Robufy — сайт + админ-панель

## Запуск

```bash
node server.js
```

Затем откройте:
- Магазин: http://localhost:3000
- Админ-панель: http://localhost:3000/admin

## Пароль

Пароль АДМИНКИ хранится ТОЛЬКО в файле `.env` (ключ `ADMIN_PASSWORD`) или в переменной
окружения. В HTML/CSS/JS, которые уходят в браузер, пароля нет. Проверка выполняется
на сервере, а браузеру выдаётся только подписанная HttpOnly cookie-сессия.

### Сменить пароль

Откройте `.env` и поменяйте значение `ADMIN_PASSWORD`. Перезапустите сервер. Код менять не нужно.

### Важно

- `.env` не коммитьте в git (уже добавлен в `.gitignore`).
- Для боевого сервера обязательно включите HTTPS и задайте свой `SESSION_SECRET`.
- Данные в админке сейчас демонстрационные (функция `adminData()` в `server.js`) — подключите свою БД.

## Структура

```
server.js          — сервер (без зависимостей, только Node)
.env               — секреты (пароль здесь)
.env.example       — шаблон конфига
public/index.html  — магазин
public/styles.css
public/script.js
public/admin.html  — админ-панель (пароля в коде нет)
public/admin.css
public/admin.js
```
